Suggestion

Vous pouvez suggérer deux types de mini jeux, des QCMs ou des mini-jeux plus ludiques de votre création
Vous trouverez ici toutes les ressources nécessaires aux développements de ceux-ci.
Le contenu doit être approprié aux élèves de troisièmes. Le score doit toujours être visible à l'écran dans le coin en haut à gauche.
Si vous souhaitez réaliser un mini-jeu de votre création, vous devez respecter la structuration du dossier fourni en complétant le contenu de game.js en effaçant son contenu.

Une fois le code de votre mini-jeu finalisé, envoyer le à l'adresse : legendofada.contact@gmail.com
